Multi-Spring Fatigue Testing Machine

For a personal design project, I designed and modeled a multi-spring fatigue testing machine to evaluate spring durability under repeated cyclic loading. The system tests four springs simultaneously and identifies failure through permanent deformation or fracture. Proper bearing placement and guided loading were critical to simulate real-world fatigue conditions accurately.

The design incorporates gear-driven rotating shafts and bearing-supported spring housings to minimize friction and ensure repeatable force application. I used SolidWorks for CAD modeling and MATLAB to plot load cycles and analyze fatigue behavior. A Simulink model was developed to simulate dynamic loading behavior and tune input parameters.

Load paths were verified with bearing calculations and shaft deflection estimates. The system is designed to handle high cycle counts with minimal wear, and the modular setup allows quick spring swaps for multiple test iterations.

Top Skills Utilized: Mechanical Design, 3D Modeling, Test Fixture Development, Failure Analysis, Cross-Team Collaboration
